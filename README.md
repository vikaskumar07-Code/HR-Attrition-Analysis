# HR-Attrition-Analysis
Power BI dashboard analyzing HR attrition to identify key factors affecting employee turnover.

# Project Overview
Employee attrition is a major challenge for organizations as it directly impacts productivity, cost, and team stability.  
This project focuses on analyzing HR data to identify key factors contributing to employee attrition and presenting insights through an interactive Power BI dashboard.

# Tools & Technologies Used
- Power BI (Dashboard & Visualization)
- Excel / CSV (Data Source)
- Data Analytics & Business Intelligence concepts
- 
# Key KPIs
- **Total Employees:** 1,470  
- **Employees Left (Attrition):** 237  
- **Attrition Rate:** 16.1%  
- **Average Age:** 37 years  
- **Average Salary:** 6.5K  
- **Average Tenure:** 7 years  

# Analysis Performed
- Attrition by **Job Role**
- Attrition by **Age Group**
- Attrition by **Salary Slab**
- Attrition by **Department**
- Attrition trends across different employee segments

# Key Business Insights
- Highest attrition observed in the **26–35 age group**
- Employees in **lower salary slabs** show significantly higher attrition
- **Laboratory Technicians** and **Sales Executives** have the highest attrition counts
- Certain departments show consistently higher employee turnover

# Business Value
This dashboard helps HR teams:
- Identify high-risk employee groups
- Understand key drivers of attrition
- Take data-driven decisions to improve employee retention
- Design targeted HR policies and compensation strategies
